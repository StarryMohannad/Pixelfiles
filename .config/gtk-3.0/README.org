[[https://gtk.org][GTK-3.0]] is a Window making framework, this config only applies the [[https://github.com/catppuccin/gtk][Catppuccin GTK Theme]], sets the [[https://fonts.google.com/specimen/Sora][Sora font]] and sets the [[https://github.com/catppuccin/papirus-folders][Papirus Catppuccin Icon Theme]], you can edit the theme in nwg-look.
